### 關於

JTMate 購物助手 Userscript 是由 Jtmate 官方開發的瀏覽器腳本，旨在幫助用戶更方便、高效地使用 Jtmate，實現更快速、更順暢的優惠券搜尋及現金回饋體驗。

### 功能

* **自動節省** – 在結帳時即時搜尋並套用可用的最佳優惠券代碼。
* **瀏覽歷史洞察** – 輕鬆追蹤 Amazon 商品歷史，幫助做出更聰明的購買決策。
* **智能優惠提醒** – 在搜尋商品時也能收到可用優惠券通知。
* **專屬優惠** – 瀏覽全站精選高價值優惠券列表。

### 使用方法

**JTMate 購物助手 Userscript** 是瀏覽器用戶腳本。使用前，需先安裝用戶腳本管理器，例如 Tampermonkey、Violentmonkey、ScriptCat 或 Userscripts。
