### Acerca de
JTMate Shopping Assistant Userscript es un script oficial desarrollado por Jtmate para ayudar a los usuarios a usar Jtmate de forma más cómoda y eficiente, ofreciendo una experiencia de búsqueda de cupones y cashback más rápida y fluida.

### Funciones
-   **Ahorro automático** - Encuentra y aplica al instante los mejores códigos de cupón disponibles al pagar.
-   **Información del historial de navegación** - Sigue fácilmente el historial de productos de Amazon para tomar decisiones de compra más inteligentes.
-   **Recordatorios inteligentes de cupones** - Recibe avisos cuando los productos tengan cupones disponibles, incluso mientras buscas.
-   **Ofertas exclusivas** - Accede a una lista curada de cupones de alto valor en todo el sitio.

### Cómo usar
**JTMate Shopping Assistant Userscript** es un userscript para navegador. Antes de usarlo, debes instalar primero un gestor de userscripts como Tampermonkey, Violentmonkey, ScriptCat o Userscripts.
