### 概要
JTMate Shopping Assistant Userscript は、Jtmate が公式に開発したスクリプトです。Jtmate をより便利かつ効率的に利用できるようにし、クーポン検索とキャッシュバックをより速くスムーズに体験できます。

### 機能
-   **自動節約** - チェックアウト時に利用可能な最適なクーポンコードを即座に見つけて適用します。
-   **閲覧履歴のインサイト** - Amazon 商品の履歴を簡単に追跡し、より賢い購買判断に役立てます。
-   **スマートクーポン通知** - 検索中でも、商品で利用可能なクーポンがあれば通知を受け取れます。
-   **限定オファー** - サイト全体から厳選された高価値クーポンの一覧にアクセスできます。

### 使い方
**JTMate Shopping Assistant Userscript** はブラウザ用ユーザースクリプトです。利用する前に、Tampermonkey、Violentmonkey、ScriptCat、Userscripts などのユーザースクリプトマネージャーをインストールする必要があります。
