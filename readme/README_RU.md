### О проекте
JTMate Shopping Assistant Userscript - это официальный скрипт, разработанный Jtmate, чтобы пользователи могли использовать Jtmate удобнее и эффективнее, получая более быстрый и плавный опыт поиска купонов и кэшбэка.

### Возможности
-   **Автоматическая экономия** - Мгновенно находит и применяет лучшие доступные купоны при оформлении заказа.
-   **Аналитика истории просмотра** - Легко отслеживайте историю товаров Amazon, чтобы принимать более взвешенные решения о покупке.
-   **Умные напоминания о купонах** - Получайте уведомления о доступных купонах на товары даже во время поиска.
-   **Эксклюзивные предложения** - Получайте доступ к подборке самых выгодных купонов по всему сайту.

### Как использовать
**JTMate Shopping Assistant Userscript** - это пользовательский скрипт для браузера. Перед использованием необходимо установить менеджер пользовательских скриптов, например Tampermonkey, Violentmonkey, ScriptCat или Userscripts.
