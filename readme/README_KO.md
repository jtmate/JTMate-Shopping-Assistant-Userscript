### 소개
JTMate Shopping Assistant Userscript는 Jtmate가 공식적으로 개발한 스크립트로, 사용자가 Jtmate를 더 편리하고 효율적으로 사용할 수 있도록 도와 더 빠르고 매끄러운 쿠폰 검색 및 캐시백 경험을 제공합니다.

### 기능
-   **자동 절약** - 결제 단계에서 사용 가능한 최적의 쿠폰 코드를 즉시 찾아 적용합니다.
-   **브라우징 기록 인사이트** - Amazon 상품 이력을 쉽게 추적해 더 현명한 구매 결정을 내릴 수 있습니다.
-   **스마트 쿠폰 알림** - 검색 중에도 상품에 사용 가능한 쿠폰이 있으면 알림을 받습니다.
-   **독점 딜** - 사이트 전체에서 엄선한 고가치 쿠폰 목록에 접근할 수 있습니다.

**JTMate Shopping Assistant Userscript**는 브라우저 사용자 스크립트입니다. 사용하기 전에 Tampermonkey, Violentmonkey, ScriptCat 또는 Userscripts 같은 사용자 스크립트 매니저를 먼저 설치해야 합니다.
